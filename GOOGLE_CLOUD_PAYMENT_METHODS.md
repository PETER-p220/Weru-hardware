## ğŸ“‹ Step-by-Step: Using Card for Google Cloud

### Step 1: Prepare Your Card
- [ ] Card is activated
- [ ] Online/international payments enabled
- [ ] At least $1 balance (for verification)
- [ ] Know your card details (number, expiry, CVV)

### Step 2: Add to Google Cloud
1. Go to [Google Cloud Console](https://console.cloud.google.com)
2. Navigate to **Billing > Payment Methods**
3. Click **"Add Payment Method"**
4. Select **"Credit or Debit Card"**
5. Enter card details:
   - Card number
   - Expiry date (MM/YY)
   - CVV (3 digits on back)
   - Cardholder name
   - Billing address (Tanzania address is fine)
6. Click **"Save"**

### Step 3: Verification
- Google charges $1 (immediately refunded)
- This verifies card is valid
- Takes 1-5 minutes

### Step 4: Set Up Billing Alerts
1. Go to **Billing > Budgets & Alerts**
2. Create budget: $5-10/month
3. Set email alerts
4. You'll get warned before any charges
