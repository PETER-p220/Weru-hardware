## ğŸ“‹ Step-by-Step Setup from Tanzania

### Step 1: Prepare Your Card
- Get Visa/Mastercard debit card from bank

### Step 4: Add Payment Method
1. Click "Add Payment Method"

4. Card is now verified

### Step 5: Enable APIs
1. Enable Maps JavaScript API
2. Enable Geocoding API
3. Create API key

### Step 6: Set Up Billing Alerts
1. Go to Billing > Budgets & Alerts
2. Create budget: $10/month
3. Set email alerts
4. You'll get warned before any charges
